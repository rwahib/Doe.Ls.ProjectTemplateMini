Overview

 

CLIPS uses a Word template to structure content, and cascading style sheets (CSS) to style that content.

 

CLIPS is the Curriculum & Learning Innovation Publishing System. It converts Word documents into web pages. It does this through highly structured documents, file naming and folder structures. Templates and CSS (Cascading Style Sheets) provide the ability for variety in look and feel for the published web pages.

It is comprised of 2 major components:

  • CLIPS Word Add-in – used for creating and previewing documents

  • CLIPS Publisher – for publishing whole or parts of a project - http://clips.det.nsw.edu.au/
